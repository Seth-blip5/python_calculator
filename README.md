# Python Calculator
A simple command-line calculator built with Python.
## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Exponentiation
- Handles division by zero
- Allows multiple calculations in one session
## Technologies Used
- Python 3
## How to Run
1. Clone the repository:
```bash
git clone https://github.com/Seth-blip5/python_calculator.git
```
2. Open the project folder.
3. Run the program:
```bash
python calculator.py
```
## Example
```
===== Python Calculator =====
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Modulus
6. Exponentiation
Enter your preferred operation(1/2/3/4/5/6):
```
